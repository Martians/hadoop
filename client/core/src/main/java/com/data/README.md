
# 需求
1. 并发度：
    1. 多线程
    1. 多机器
2. 状态汇报：
    1. P99状态统计
    1. 错误执行统计
    1. 远程执行，定期汇报当前状态
3. 可配置性：
    1. 动态加载执行类
    2. 每个类读取自己的配置
3. 随机数生成
    1. 生成顺序数据
    1. 加快生成效率
    1. 后续随机read可以在已经随机的数据中读取

# 功能
1. 解析配置
    1. 配置文件和命令行共同解析，命令行优先级更高
    2. 配置定义，每个测试类定义自己的参数集合
    3. 可以定义参数之间的互斥关系
    4. 启动后打印所有相关配置
2. 日志管理
    1. 本机测试、全局测试日志
    2. 不同类型的测试
    3. 根据测试时间生成不同的日志文件
